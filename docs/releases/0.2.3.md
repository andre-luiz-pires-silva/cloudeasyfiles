# CloudEasyFiles 0.2.3 Release Notes

`0.2.3` adds file preview support to CloudEasyFiles, making it easier to inspect cloud-stored content before downloading or changing it.

## Highlights

- File preview panel for supported objects in the explorer
- Provider-aware preview workflow for AWS S3 and Azure Blob Storage content
- Clear loading, empty, and error states while preview content is being prepared
- Preview access integrated into the existing navigation and selection experience
- Expanded quality control with new unit tests for the backend and frontend
- Improved pipeline checks to help preserve software quality in future changes

## Known Limitations

- Preview support is intentionally limited to file types the app can safely display inline
- Large or unsupported files still require download before local inspection
- Preview behavior depends on provider access permissions and object availability
