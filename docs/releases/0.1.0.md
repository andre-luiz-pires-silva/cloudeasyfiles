# CloudEasyFiles 0.1.0 Release Notes

`0.1.0` is the first functional AWS-compatible release of CloudEasyFiles.

## Highlights

- AWS connection management with saved credentials
- S3 bucket and object browsing with incremental loading
- Local filter, tracked downloads, and `Download As`
- AWS upload, folder creation, delete, restore, and storage-class change flows
- Local cache awareness, transfer monitoring, and cancelation

## Known Limitations

- Azure support is not included in this release
- Upload, restore, and storage-class changes are AWS-only
- The current upload implementation has no local queue or explicit concurrency cap
- Release packaging and publication are manual

## Next Milestone

The next work cycle focuses on Azure support for the workflows already established in the AWS-first release.
